Theory of sets

Sets

  • In a space XXX a set EEE is well defined if there is a rule which determines,for each element (or point) xxx in XXX, whether or not it is in EEE. We write x∈Ex\in Ex∈E (read ‘xxx belongs to EEE’) whenever xxx is an element of EEE, and the negation of this statement is written x∉Ex\notin Ex∈/E. Given two sets EEE, FFF we say that EEE is contained in FFF, or EEE is a subset of FFF, or FFF contains EEE and write E⊂FE \subset FE⊂F if every element xxx in EEE also belongs to FFF. If E⊂FE \subset FE⊂F and there is at least one element in FFF but not in EEE, we say that EEE is a proper subset of FFF.

  • Two sets EEE, FFF are equal if and only if they contain the same elements; i.e. if and only if E⊂FE \subset FE⊂F and F⊂EF \subset EF⊂E. In this case we write E=FE = FE=F. This means that if we want to prove that E=FE = FE=F we must prove both x∈E⇒x∈Fx \in E \Rightarrow x \in Fx∈E⇒x∈F and x∈F⇒x∈Ex \in F \Rightarrow x \in Ex∈F⇒x∈E.

  • We frequently have to consider sets of sets, and occasionally sets of sets of sets. It is convenient to talk of classes of sets and collections of classes to distinguish these types of set, and we will use italic capitals A,B,...A,B,...A,B,... for sets, script capitals A,B,...\mathcal{A},\mathcal{B},...A,B,... for classes and Greek capitals Δ,Γ,...\Delta, \Gamma,...Δ,Γ,... for collections. Thus C∈CC \in \mathcal{C}C∈C is read ‘the set CCC belongs to the class C\mathcal{C}C’; and A⊂B\mathcal{A} \subset \mathcal{B}A⊂B means that every set in the class A\mathcal{A}A is also in the class B\mathcal{B}B.

Cartesian product

Given two sets EEE, FFF we define the Cartesian (or direct) product E×FE \times FE×F to be the set of all ordered pairs (x, y) whose first element x∈Ex\in Ex∈E and whose second element y∈Fy \in Fy∈F. This clearly extends immediately to the product E1×E2×...×EnE_1 \times E_2 \times ... \times E_nE1​×E2​×...×En​ of any finite number of sets. In particular it is immediate that Rn\mathbb{R}^nRn, Euclidean nnn-space, is the Cartesian product of nnn copies of R\mathbb{R}R. For an infinite indexed class {Ei,i∈I}\{ E_i,i\in I\}{Ei​,i∈I} of sets, the
product ∏i∈IEi\prod_{i\in I} E_i∏i∈I​Ei​ is the set of elements of the form {ai,i∈I}\{ a_i,i\in I\}{ai​,i∈I} with ai∈Eia_i \in E_iai​∈Ei​ for each i∈Ii\in Ii∈I.

Semi-algebra

Today we define functions about some classes of subsets.

P(Ω)⊇S\mathcal{P}(\Omega) \supseteq \mathcal{S} P(Ω)⊇S

P\mathcal{P}P represents all subsets of.
A class S\mathcal{S}S is a semi-algebra if holds three properties:

  • 包含全集

    Ω∈S\Omega \in \mathcal{S} Ω∈S

  • 有限交闭合

    A,B∈S⇒A∩B∈SA,B \in \mathcal{S} \Rightarrow A \cap B \in \mathcal{S} A,B∈S⇒A∩B∈S

  • 补集可以有限分解

    ∀A∈S⇒∃E1,...,Eh∈Ss.t.Ac=∑j=1hEj\forall A \in \mathcal{S} \Rightarrow \exist E_1,...,E_h \in \mathcal{S} \quad \\ s.t. \quad A^c = \sum_{j=1}^h E_j ∀A∈S⇒∃E1​,...,Eh​∈Ss.t.Ac=j=1∑h​Ej​

Here ∑\sum∑ refers to disjoint ∪\cup∪, which means EiE_iEi​ are disjoint with each other. In fact, owing to property 2, the ∑\sum∑ can be transformed into ∪\cup∪. But for convenience in some proof, the property 3 use the ∑\sum∑.

Algebra

Def 2:\textbf{Def 2:}Def 2:

A⊆P(Ω)\mathcal{A} \subseteq \mathcal{P}(\Omega) A⊆P(Ω)

A class A\mathcal{A}A is an algebra if the following holds；

  • 包含全集

    Ω∈A\Omega \in \mathcal{A} Ω∈A

  • 有限交闭合(等价于有限并闭合)

    A,B∈A⇒A∩B∈AA,B \in \mathcal{A} \Rightarrow A \cap B \in \mathcal{A} A,B∈A⇒A∩B∈A

  • 补运算闭合(更加严格)

    A∈A⇒Ac∈AA \in \mathcal{A} \Rightarrow A^c \in \mathcal{A} A∈A⇒Ac∈A

Obs:\textbf{Obs:}Obs:

Aalgebra⇒Asemi-algebra\mathcal{A} \quad \text{algebra} \Rightarrow \mathcal{A} \quad \text{semi-algebra} Aalgebra⇒Asemi-algebra

Sigma-algebra

Def 3:\textbf{Def 3:}Def 3:

F⊆P(Ω)\mathcal{F} \subseteq \mathcal{P}(\Omega) F⊆P(Ω)

A class F\mathcal{F}F is an σ\sigmaσ-algebra if the following holds；

  • 包含全集

    Ω∈F\Omega \in \mathcal{F} Ω∈F

  • 可数交闭合(更加严格)

    Aj∈F⇒∩Aj∈FA_j \in \mathcal{F} \Rightarrow \cap A_j \in \mathcal{F} Aj​∈F⇒∩Aj​∈F

  • 补运算闭合

    A∈F⇒Ac∈FA \in \mathcal{F} \Rightarrow A^c \in \mathcal{F} A∈F⇒Ac∈F

Obs:\textbf{Obs:}Obs:

Aσ-algebra⇒Aalgebra\mathcal{A} \quad \sigma\text{-algebra} \Rightarrow \mathcal{A} \quad \text{algebra} Aσ-algebra⇒Aalgebra
